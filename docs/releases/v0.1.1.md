# Flow v0.1.1 Release Notes (2026-03-05)

Patch release for `flow.gocools` focused on reliability and observability consistency.

## Highlights

- Hardened CloudWatch metrics/logs collectors with bounded retries, jittered backoff, and retry/throttle metrics.
- Normalized HTTP route labels (`route` over raw path) to reduce metric and trace cardinality.
- Made ingestion integration harness deterministic for stable CI behavior and reproducible debugging.
- Enabled release workflow artifact upload for cross-platform binaries.
