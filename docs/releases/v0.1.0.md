# Flow v0.1.0 Release Notes (2026-03-05)

Initial public MVP release for `flow.gocools`.

## Highlights

- API bootstrap with health/readiness and telemetry endpoints.
- CloudWatch metrics and logs ingestion pipelines.
- Telemetry correlation graph model and incident timeline API.
- Storage abstraction with local and S3 backends.
- AWS STS AssumeRole runtime configuration and startup validation.
- Observability instrumentation (`/metrics`, tracing) and integration harness.
